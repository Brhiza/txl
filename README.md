# 你是不是同性恋？

基于 `mingyu-core` 八字排盘的娱乐向速断网页。

输入名字、性别、出生日期、出生时辰后，直接给出：

- 取向：gay / les / 双性恋 / 不是
- 0 或 1
- 是否深柜
- 判定理由
- 结果大标题

> 仅供娱乐，不构成任何医学、心理或现实鉴定结论。

## 本地运行

```bash
npm install
npm run dev
```

浏览器打开终端里给出的本地地址，通常是：

```text
http://127.0.0.1:5173
```

## 构建

```bash
npm run build
```

产物在 `dist/`。

## 功能说明

1. 用 `mingyu-core` 计算完整八字盘
2. 综合配偶星、桃花神煞、十二长生、日主阴阳旺衰、格局、十神透藏做多因子判定
3. 结果区分：
   - 不是同性恋
   - 同性恋（男 gay / 女 les）
   - 双性恋
4. 同性恋/双性恋时继续判定：
   - `0` / `1`
   - 是否深柜
5. 可把当前结果保存成分享图（按名字命名；图片不展示八字隐私信息）
6. 底部提供“更多测试”入口：`https://sydf.cc`

## 目录

```text
index.html
src/
  main.js      # 页面交互与保存图片
  analyze.js   # 八字判定算法
  style.css    # 样式
package.json
```

## 部署

可部署到 Cloudflare Pages 等静态托管：

1. `npm run build`
2. 上传 / 发布 `dist`
3. 绑定自定义域名，例如 `txl.sydf.cc`

## 说明

- 未知时辰不要乱填；时辰会改变时柱，从而影响结果
- 同一组输入结果固定
- 保存图片在浏览器本地生成，不上传服务器
